# Cafe BOS - Order Coffee System

Sistem manajemen kasir dan absensi untuk Cafe BOS.

## Fitur
- Pemesanan Kopi
- Sistem Absensi (Face Detection)
- Manajemen Inventaris
- Laporan Penjualan (Excel/PDF)

## Cara Menjalankan
1. Pastikan Python terinstal.
2. Jalankan `server.py` atau `app.py`.
3. Gunakan `jalankan_kasir.bat` untuk kemudahan akses di Windows.
